# Customer Analytics Dashboard (Power BI)

## Overview
This project analyzes customer data to uncover insights on income, spending behavior, and customer value.

## Tools
- Power BI
- Excel / CSV

## Key Insights
- Higher income customers generate higher customer value
- Customer value increases with engagement (membership years)
- Adults form the largest customer segment

## Dashboard Features
- KPI Cards (Income, Spending Score, Customer Value)
- Scatter Plot (Income vs Customer Value)
- Line Chart (Membership vs Purchase Frequency)
- Pie Chart (Customer Distribution)
- Interactive Filters
